给博客减肥

Posted on 2017-05-09

Github pages的空间是有限的。以后博客要是写多了，说不定哪一天空间就会不够用。

刚想到一个给博客减肥的办法。分四步：
1.去Github删除博客的Repo（不用怕，因为最新博客数据在硬盘上有存档）；
2.删除硬盘上Hexo文件夹中的.deploy_git文件夹；
3.重建同名Repo
4.Hexo d

这样一来那些重复的版本记录就被清除了。本来博客就只有最新版有用，几乎不会用到版本记录信息，所以该减肥法几乎没有副作用，哈哈

junit note 170508

Posted on 2017-05-08

basic JUnit annotations

@BeforeClass – 最初运行
@Before – @Test 之前运行
@Test – test case
@After – @Test 之后运行
@AfterClass – 最后运行
引包举例：
import org.junit.Before;
import org.junit.Test;
import static org.junit.Assert;

Class Assert

Assert类下面有最关键的一组静态方法，如：
fail(String msg)
可自己实现一个validate方法，如果false了，就
fail(“Should not happen”);

其它还有
assertEquals(obj1, obj2)
assertArrayEquals(array1, array2)
assertTrue(boolean condition)
assertFalse(boolean condition)
assertNull(obj)
assertNotNull(obj)
assertSame(obj1, obj2)
assertNotSame(obj1, obj2)

ノルウェイの森

でね、いまピンときたの！この本のfirst-person narratorは「The Great Gatsby」のとそっくりだ！

これで、「The Great Gatsby」をも読みかえしたくなったなぁ！

deploy后需要等待

Posted on 2017-05-02

deploy后由于缓存（本地的 and 网上的）原因，网站要么没更新，要么只更新了一部分，看起来问题多多。
这类问题不必放在心上。只要hexo s本地验证没问题就没问题。
比如昨晚我在本地调得好好的，而deploy后，不光blog背景是旧的，显示代码段的内容还莫名多出了个横向滚动条。但今早再看，一切都正常了！
